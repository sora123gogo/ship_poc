## Ship Detection & Tracking PoC

港湾監視映像における船舶検出・追跡システムの PoC デモです。

YOLOv8 + ByteTrack を用いて、長時間映像から船舶の検出・追跡・滞留分析を行います。
昼夜環境に適応した推論処理および統計データ出力に対応しています。


## 機能ハイライト

- 船舶検出：YOLOv8 による物体検出

- マルチオブジェクト追跡：ByteTrack による ID 付与・継続追跡

- 昼夜適応処理：輝度解析による自動 Day / Night 切替

- フレーム抽出：長時間動画からの自動フレーム生成

- 可視化デモ：追跡結果の動画出力（平滑化対応）

- データ出力：JSONL / CSV 形式による解析用データ生成

- 軌跡統合処理：分断された追跡 ID の後処理統合（オプション）



## クイックスタート

### 仮想環境の作成と有効化
```powershell
conda create -n ship_poc python=3.10 -y
conda activate ship_poc
```
### 依存関係のインストール
```bash
pip install -r requirements.txt
```

### 映像データの配置
解析対象の港湾映像を以下に配置してください：
```bash
data/raw/
```

### 注意事項
可視化平滑化処理は動画表示のみに適用され、元データには影響しません

検出精度は映像品質に依存します

実運用環境では RTSP ストリーミング入力を推奨します
